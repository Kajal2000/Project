# Project

Step 1: Install
  * npm init 
  * npm install express --save
  * npm install mysql
  * npm install knex --save
  * npm install jsonwebtoken
  * npm install -g nodemon
  * npm install --save-dev nodemon


Step 2: Uses
  * Express Framework
  * jwt for authentication
  * knex query
  * Mysql
  
  
Step 3: Run 
  * nodemon server.js
